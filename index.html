Lambda is the Future

It is Extremely Cheap and Very Scalable

Serverless computing completely changes the game. Through the tutelage of Dr. Kettles we have learned how to dynamically change a website using AWS API Gateways and AWS Lambda. The website is hosted in a S3 bucket, and we use AWS Code Pipeline to deploy new changes. It is all very scalable as Lambda will just charge us more for the increased execution time as more people visit the site.
